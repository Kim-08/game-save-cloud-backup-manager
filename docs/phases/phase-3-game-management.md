# Phase 3: Game Management

## Goal

Allow users to manage games and their backup configuration.

## Tasks

- Add game list UI.
- Add create/edit/delete game flows.
- Store game entries in local JSON.
- Validate EXE path, save folder path, rclone remote, and cloud folder.
- Add per-game enabled/disabled setting.

## Exit criteria

- Users can persistently manage game entries.
